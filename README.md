Analisis de señales usando transformador 
# Análisis de Señales con Transformada de Fourier

## Descripción
Este proyecto analiza señales en el dominio del tiempo y la frecuencia utilizando Python.

## Señales analizadas
- Señal senoidal
- Pulso rectangular
- Función escalón

## Herramientas
- Python
- NumPy
- Matplotlib

## Objetivo
Aplicar la Transformada de Fourier para visualizar el comportamiento de señales.

## Ejecución
Ejecutar el archivo:
python main.py
